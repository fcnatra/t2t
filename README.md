"# T2T" 

Esta prueba de concepto es una app para android
Tendrá una IU básica
La app, con opciones de la IU podrá conectarse a puertos remotos
También escuchar peticiones por un puerto
Mostrará en pantalla la IP actual del dispositivo en todo momento
Al cambiar la IP, actualizará el valor en pantalla y mostrará una notificación
